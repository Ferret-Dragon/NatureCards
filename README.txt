
📋 To run the application:
   1. Activate virtual environment: source venv/bin/activate
   2. Run the app: python app.py
   3. Open http://127.0.0.1:5000 in your browser

  The website includes:
  - 🌿 Nature-themed UI with earthy colors and plant emojis
  - 📚 Deck organization for different class subjects
  - 🔄 Interactive study mode with keyboard shortcuts
  - 📱 Responsive design for mobile devices
  - 🎯 Clean, intuitive navigation

  The study mode supports clicking to flip cards, keyboard navigation
  (spacebar/enter to flip, arrows to navigate), shuffling, and progress
  tracking.
